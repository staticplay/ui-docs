Geting Started

In your command line application, you will want to install the necessary tools to start working on the UI framework. Make sure your other projects are at the same level for synced releases.

Install

SHELL
 npm i -g gulp
 npm i
 gulp watch

Setup

Make sure to add these things to your layout file

HTML
 <html class="s-app">

 <head>
   <meta name="viewport" content="width=device-width, initial-scale=1">
   <link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Source+Sans+Pro:300,400,400italic,600,700">
   <link rel="stylesheet" href="/path/to/app.css">
 </head>

 <body class="t-default">
   <div class="s-app-container">

Media

Responsive breakpoints for styling for different devices and groups of devices.

Handhelds

Only Small Devices @media (--sm) { }

Small plus larger devices @media (--sm-up) { }

Tablets

Only Medium Devices @media (--md) { }

Medium plus larger devices @media (--md-up) { }

Medium plus smaller devices @media (--md-down) { }

Computers

Only Large Devices @media (--lg) { }

Large plus larger devices @media (--lg-up) { }

Large plus smaller devices @media (--lg-down) { }

Printers

Simply put, @media print { }

Classes

You will also notice some classes are appended with @sm, @md, @lg, and @print for the same purposes.

See syntax/responsive-suffixes for more info.

Bar

Use the class .c-bar on a <div> or <header> to get an application style top bar.

HTML
 <header class="c-bar">
    <nav>
      <ul>
        <li><h1 class="o-brand">Title</h1></li>
        <li><i class="o-icon-right-dir"></i><h2>Item</h2></li>
      </ul>
    </nav>
    <nav>
      <ul>
        <li><a href="#"><i class="o-icon-help-circle"></i></a></li>
      </ul>
    </nav>
 </header>
