Media

Responsive breakpoints for styling for different devices and groups of devices.

Handhelds

Only Small Devices @media (--sm) { }

Small plus larger devices @media (--sm-up) { }

Tablets

Only Medium Devices @media (--md) { }

Medium plus larger devices @media (--md-up) { }

Computers

Only Large Devices @media (--lg) { }

Large plus larger devices @media (--lg-up) { }

Printers

Simply put, @media print { }

Classes

You will also notice some classes are appended with @sm, @md, @lg, and @print for the same purposes.

See syntax/responsive-suffixes for more info.

Bar

Use the class .c-bar on a <div> or <header> to get an application style top bar.

HTML
 <header class="c-bar">
    <div class="c-bar__title">
      <h1 class="o-heading o-heading--company o-heading--brand">Title</h1>
      <div>Item</div>
    </div>
    <div class="c-bar__actions">
      <div><a href="#"><i class="o-icon-help-circle"></i></a></div>
    </div>
 </header>

Accessibility

Customize

CSS
 :root {
  --button-background: var(--orange-600);
  --button-background-hover: var(--orange-500);
  --button-color: contrast(var(--button-background));
  --button-disabled-opacity: .35;
  --button-font-size: 1.125rem;
  --button-font-size-sm: calc(var(--button-font-size) * .75);
  --button-font-size-lg: calc(var(--button-font-size) * 1.25);
  --button-margin: var(--global-margin);
  --button-padding: .7em 1.5em;
  --button-round: var(--global-rounded);
 }
